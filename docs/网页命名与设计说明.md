# 网页命名与设计说明

## 1. 主推荐名称

中文名：衣橱志  
英文名：Atelier Archive

### 命名理由
1. “衣橱”直接对应产品核心场景，语义清晰。
2. “志”有记录、整理、归档的意味，不会像普通工具名那样太硬。
3. `Atelier Archive` 带一点工作室和档案库的气质，适合“个人衣物管理系统”的定位。
4. 中英文都适合做网页标题、Logo 字样和后续品牌延展。

## 2. 备选名称

### 方案 A
中文名：衣柜册  
英文名：Wardrobe Ledger

适合更偏管理工具、台账感更强的方向。

### 方案 B
中文名：衣序  
英文名：Closet Order

更简洁，强调秩序感和整理感，但品牌识别度略弱。

### 方案 C
中文名：着装档案  
英文名：Style Archive

更偏穿搭和风格表达，弱化“收纳管理”的感觉。

## 3. 当前网页设计方向

本轮原型采用“衣橱志 / Atelier Archive”作为默认名称，并以此建立视觉方向：

1. 气质上偏“个人衣物工作台”，不是电商，也不是记账后台。
2. 颜色采用纸张、苔绿色、赭石色的组合，降低科技产品的冰冷感。
3. 版式上强调信息秩序、卡片化管理和编辑感，方便后续扩展衣柜系统功能。

## 4. 首版原型包含内容

1. 产品总览区
2. 我的衣柜模块
3. 新增衣物录入模块
4. 自动穿搭模块
5. 季节整理模块
6. 收纳地图模块
